tokei - tokei is a line count tool written in Rust
  • Count the code of the project
    • sh
    x tokei data.yml
    x tokei data.yml
  • Count the code of the project
    • sh
    x tokei ./
    x tokei ./
  • Sort languages based on column
    • sh
    x tokei --sort code
    x tokei --sort code
  • Ignore statistics json files
    • sh
    x tokei --exclude '*.json'
    x tokei --exclude '*.json'
  • Only counted json files
    • sh
    x tokei --type json
    x tokei --type json
  • Output statistical information in yaml format
    • sh
    x tokei --type json -o yaml
    x tokei --type json -o yaml
